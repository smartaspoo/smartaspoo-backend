# SMART ASPOO APP

## Requirement

- PHP8.1
- MYSQL
- Nodejs

## Cara Install

`composer install`
`npm install`

## Run Aplikasi

`npm run dev`
`php artisan serve`

NPM cuma untuk build viewnya doang, soalnya pakai vue. Server tetep pakai php artisan

## Run Laragon (server)

`php -S localhost:7000 -t smartaspoo-backend/public`
`ngrok http localhost:7000`
